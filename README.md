# Makrees — Global Growth Agency Website

Built with React + Vite. Full 5-page SPA: Home, Services, Technology, About, Contact.

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

## Deploy on Vercel
Connect this repo to Vercel — it auto-detects Vite. Build command: `npm run build`, Output: `dist`.
